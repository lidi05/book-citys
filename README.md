>项目说明：
>此项目是一个书城项目，页面精美，代码简洁，用了的人都对此项目叫好，可以说此项目深得民心，是民心所向。性能方面来说也是非常棒的，在这里拿别的书城项目与此项目作对比，别的书城加载一章小说的流量这个项目可以加载两章，这意味着在线观看本项目的小说可以为观众节省一半的流量。这一点是别的书城做不到的。所以希望有更多的人喜欢这个项目，为这个项目点赞。
>采用技术：
> 本项目所采用的技术：handlebars,gulp,jquery,swiper,bscroll等等
